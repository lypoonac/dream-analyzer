# 💭 AI Dream Analyzer - Academic Project

[![Streamlit App](https://static.streamlit.io/badges/streamlit_badge_black_white.svg)](https://your-app-name.streamlit.app)

An AI-powered dream analysis tool that combines traditional Zhou Gong dream interpretation with modern AI emotion and stress analysis.

## ✨ Features
- **🤖 Dual AI Model System**: Uses two Hugging Face models
- **🎯 Zhou Gong Integration**: Traditional dream symbol interpretation
- **📊 Stress Assessment**: Calculates stress levels (1-10 scale)
- **💡 AI-Generated Recommendations**: Personalized wellness advice
- **📈 Analytics Dashboard**: Track dream patterns over time
- **💾 Export Functionality**: Save analyses as CSV

## 🚀 Quick Start

### Local Installation
```bash
# Clone repository
git clone https://github.com/yourusername/dream-analyzer.git
cd dream-analyzer

# Run setup script
python setup.py

# Install dependencies
pip install -r requirements.txt

# Run the app
streamlit run app.py
